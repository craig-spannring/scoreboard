Overview of the CRG Scoreboard Project

Project Description

The CRG Scoreboard is a platform independent web based application to track score, time, penalties, and line ups. It is assumed that maintainers of this software will be familiar with roller derby and the structure of play.

Design Goals
  • Decrease cognitive load on NSOs.
  • Cross Platform
  • Able to run on older hardware

Internal Game Logic

User Interface

The user interface is web browser based.

Explain path messages from UI take to go through com.carolinarollergirls.scoreboard.jetty.WS.WebSocketServlet.ScoreBoardWebSocket.onMessage and eventually down to the execute() method in classes such as GameImpl.
